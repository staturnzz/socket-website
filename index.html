## Installing (With PC) ```bash 1. Download the IPA file from the button above 2. Sideload it using Sideloadly or a similar tool 3. Go to Settings > General > Profiles & trust the app 4. Open Socket on your home and press jailbreak ```
## Installing (Without PC) ```bash 1. On your device go to jailbreaks.app/legacy.html 2. Click on the option for installing Socket 3. Press on "Install" when you get a promt asking 3. Go to Settings > General > Profiles & trust the app 4. Open Socket on your home and press jailbreak ```
## Information and Notes ```bash - The default package manager is Zebra NOT Cydia - Sandbox containers are not destoryed Learn More - You can check out the source code on my GitHub - If you have issues please report it on the GitHub ``` *for a more detailed guide please visit ios.cfw.guide*
